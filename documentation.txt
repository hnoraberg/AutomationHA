Documentation for GoodReads Scraping Automation

Prerequisites for running scripts:
Install scrapy:
    pip install scrapy
Install Google client library:
    pip install --upgrade google-api-python-client google-auth-httplib2 google-auth-oauthlib
Install Pandas:
    pip install pandas

Download 'home_assignment_automation_HN' directory
Start 'scheduler' script in IDE
Create a project containing 'Google_spreadsheet_and_forms' script in Google Script:
    Tools -> Script Editor -> Edit -> Current Project's Triggers -> Add Trigger
    Set a weekly trigger
    Run script
